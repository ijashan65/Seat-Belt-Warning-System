# Seat Belt Warning System using Arduino

## Objective
Detect whether the seat belt is fastened and provide a warning using LED and buzzer when the belt is not worn.

## Components Used
- Arduino Uno
- Push Button
- LED
- Buzzer
- Resistors
- Breadboard
- Jumper Wires

## Working Principle
The push button simulates the seat belt sensor.

- Seat belt fastened → LED ON
- Seat belt not fastened → Buzzer ON and warning indication
